## Getting Help

### When Stuck

1. **Check Official Docs**: https://developer.android.com
2. **Search Stack Overflow**: Tag: `android`, `kotlin`, `jetpack-compose`
3. **Read Architecture Guide**: In this project
4. **Review Sample Code**: From Google samples
5. **Ask in Communities**: r/androiddev, Android Developers forums

### Common Issues

**Gradle Issues**:
- [Gradle Troubleshooting Guide](https://developer.android.com/studio/build)
- Invalidate cache and restart Android Studio
- Delete .gradle folder and sync again

**Compose Issues**:
- [Compose FAQ](https://developer.android.com/jetpack/compose/resources)
- Check Compose version compatibility
- Clear build cache

**API Issues**:
- [Retrofit Docs](https://square.github.io/retrofit/)
- Check request/response format
- Verify endpoint URLs

## Documentation Standards

### This Project Uses

- **Code Documentation**: KDoc comments
- **Architecture Docs**: Markdown files
- **API Docs**: JavaDoc/KDoc
- **User Guides**: Markdown with examples
- **Comments**: Inline for complex logic

### When Contributing

Follow these standards:
- Document public functions with KDoc
- Add comments for complex algorithms
- Update README for new features
- Create/update relevant markdown docs
- Include examples where helpful
